# Comprehensive Construction Audit

Perform a comprehensive end-to-end construction audit for a new rules engine whose current implementation has unclear prerequisites, speculative abstractions, oversized routines, primitive flags, tangled loops, unchecked inputs, inconsistent error policy, hidden table encodings, mixed-responsibility modules, stale comments, guess-driven debugging, premature optimization, and happy-path-only tests.

Cover prerequisites and design decisions, routines, variables and data types, control flow, defensive programming, errors, tables, class and module shape, contracts, comments and standards, incremental construction, debugging, integration, performance, tooling, tests, forbidden patterns, and final construction review. Keep recommendations concrete and ordered by risk.
