# Comprehensive Local Code Quality Audit

Perform a comprehensive end-to-end code-quality audit of a notification module that has vague names, boolean modes, long mixed-phase functions, flow-narrating comments, hidden cache mutation, null sentinels, framework DTOs in business logic, broad exception handling, shared mutable async state, brittle order-dependent tests, and repeated conditional branches.

Cover naming, functions, state and side effects, comments, formatting and structure, objects and data, classes and modules, errors, external boundaries, system construction, tests, concurrency, smells, successive refinement, change discipline, and hard stopping rules. Recommend a bounded correction order without turning the answer into a rewrite plan.
