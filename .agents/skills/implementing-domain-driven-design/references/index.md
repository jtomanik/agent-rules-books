# Implementing Domain-Driven Design Reference Index

Use `SKILL.md` alone when it resolves ordinary matched implementation work. For an explicit dispute, demonstrated implementation hotspot, or one bounded question left unresolved after applying the skill, use the table to select the smallest relevant sections of [full.md](full.md). For a complete DDD implementation audit or an explicit request for the complete Implementing Domain-Driven Design lens, read [full.md](full.md) end to end.
