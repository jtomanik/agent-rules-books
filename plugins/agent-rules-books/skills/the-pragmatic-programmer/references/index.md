Use `SKILL.md` alone when it resolves ordinary matched work. For an explicit source dispute or one bounded question left unresolved after applying the skill, use the table to select the smallest relevant sections of [full.md](full.md). For a comprehensive audit or an explicit request for the complete book lens, read [full.md](full.md) end to end.
