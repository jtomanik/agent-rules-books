Use `SKILL.md` alone when it resolves ordinary matched work. For an explicit dispute, demonstrated pattern hotspot, or one bounded question left unresolved after applying the skill, use the table to select the smallest relevant sections of [full.md](full.md). For an exhaustive enterprise application pattern audit or an explicit request for the complete lens, read [full.md](full.md) end to end.
